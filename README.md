# MultiplayerTestTask

**Unity version:** 6000.0.31f  

In the editor, the game should be started from the **Bootstrap scene**.

## Photon Setup

To start playing the game, you first need to set up a **Photon App**:

1. Go to the [Photon website](https://www.photonengine.com/) and open the **Dashboard**.  
2. Create a new application.  
3. Copy the **App ID** and paste it into **Photon Fusion Hub** (`Tools/Fusion/Fusion Hub`).  
4. Create a second application for **voice chat**.  
5. Open **Photon App Settings** (`Assets/Photon/Fusion/Resources`) and paste the voice chat **App ID** into the **App ID Voice** field.

## Build the Project

1. Go to **File → Build Profiles** and build the project.  

## Starting the Game

- After launching the game, you can (optionally) enter a **room code** and start the game.  
- The game will automatically select either **host** or **client mode**.  

## Voice Chat

- Press and hold **V** to use voice chat.

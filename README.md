# MultiplayerTestTask

Unity version 6000.0.31f. In the editor, the game should be started from the Bootstrap scene.

To start playing the game, you first need to set up a Photon App.

1. Go to the Photon websiteand open the Dashboard.
2.Create a new application.
3.Copy the App ID and paste it into Photon Fusion Hub (Tools/Fusion/Fusion Hub).
4.Create a second application for voice chat.
5.Open Photon App Settings (Assets/Photon/Fusion/Resources) and paste the voice chat App ID into the App ID Voice field.

Then go to File/Build Profiles and build the project.

After launching the game, you can (optionally) enter a room code and start the game. The game will automatically select either host or client mode.

Press and hold V to use voice chat.
